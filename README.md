# VUE-ELEMENT-PLUS-ADMIN-TS

this app template use vuenext + elementPlus + typescript + vite 

For future 面向未来的开发方案,默认配置兼容到支持 script module 方式的浏览器


## Environment 环境
- node v14+ (理论上 12.7+ 就行)
- vite v2
## Install 安装依赖
```bash
npm i
```
## Dev 运行项目
```bash
npm run dev
```
## Build 构建项目
```bash
npm run build
```
